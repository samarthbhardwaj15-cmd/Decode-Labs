# 🤖 Rule-Based AI Chatbot

## Artificial Intelligence Internship - Project 1

### Description
This project is a simple rule-based AI chatbot developed using Python. It interacts with users by responding to predefined questions and commands using conditional statements.

## Features
- Greeting responses
- Answers basic questions
- Help command
- Exit command
- Continuous conversation using a loop

## Technologies Used
- Python
- VS Code

## How to Run

```bash
python chatbot.py
```

## Learning Outcome
- if-else statements
- Loops
- User input handling
- Basic AI concepts
