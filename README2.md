# 🌸 Data Classification Using AI

## Artificial Intelligence Internship - Project 2

### Description
This project demonstrates a basic machine learning classification model using the Iris dataset and the Decision Tree algorithm.

## Features
- Load dataset
- Split training and testing data
- Train Decision Tree model
- Predict results
- Display accuracy

## Technologies Used
- Python
- Scikit-Learn
- VS Code

## Dataset
Iris Dataset

## How to Run

```bash
pip install scikit-learn
python classification.py
```

- Machine Learning Basics
- Data Classification
- Model Training
- Model Evaluation
